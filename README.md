# CloudSecure Inc. GRC Portfolio Project

## Project Overview
This repository contains a simulated GRC (Governance, Risk, and Compliance) program for **CloudSecure Inc.**, a mid-sized SaaS company (150 employees). This project demonstrates the operational implementation of **ISO 27001:2022**, while aligning with **NIST CSF 2.0**, **GDPR**, and **PCI DSS v4.0**.

## Business Context
* **Infrastructure:** AWS (Cloud-native).
* **Operations:** SaaS Platform, Mobile App, Internal Admin Portal.
* **Compliance Drivers:** GDPR (EU Customer Data), PCI DSS (Stripe payment processing).

## Repository Structure
- `/01_scope_and_context`: Defining boundaries and stakeholder interests.
- `/02_risk_management`: The core of the program (Risk Register/Treatment).
- `/03_policies_and_controls`: Framework of governance.
- `/04_compliance_mappings`: Cross-framework synchronization.
- `/05_audit_evidence`: Artifacts for certification.
- `/06_training_awareness`: Human-centric security.
- `/07_metrics_and_kpis`: Continuous monitoring data.

## Portfolio Presentation Note
This repository serves as a *proof of competency* in GRC program management. In an interview, I utilize these artifacts to explain:
1. How I translate business risk into technical controls.
2. How I map ISO 27001 requirements to day-to-day operations.
3. How I use metrics to provide "Continuous Compliance" visibility to executive leadership.
