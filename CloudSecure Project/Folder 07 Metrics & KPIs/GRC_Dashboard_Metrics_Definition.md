# GRC KPI Definitions

| KPI Name | Objective | Target | Frequency |
| :--- | :--- | :--- | :--- |
| **Phishing Click Rate** | Measure training efficacy | < 5% | Monthly |
| **Mean Time to Remediate (MTTR)** | Measure Incident Response | < 4 Hours | Quarterly |
| **Asset Coverage** | Ensure all assets in inventory | 100% | Monthly |
| **Audit Finding Closure Rate** | Measure compliance program health | 100% in 30 days | Ongoing |
