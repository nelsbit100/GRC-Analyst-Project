# ISMS Scope Statement - CloudSecure Inc.

## Version: 1.0
## Date: 2026-07-15

## 1. Scope Boundary
The Information Security Management System (ISMS) applies to all corporate infrastructure, software development, and customer-facing SaaS services of CloudSecure Inc., operated within AWS environments.

## 2. In-Scope Services
* **CloudSecure SaaS Platform:** Main application stack.
* **CloudSecure Mobile App:** iOS/Android versions.
* **Internal Admin Portal:** Management console for internal support staff.

## 3. Exclusions
* Third-party managed hardware at AWS facilities (covered by AWS SOC2 Type II reports, shared responsibility model).
